# 💰 Personal Finance Tracker

A **Streamlit-based Personal Finance Tracker** that helps users analyze bank and credit card transactions using **Machine Learning** and interactive visualizations.

## ✨ Features

* Upload transaction CSV files
* Automatic transaction categorization
* Rule-based + ML (Logistic Regression) classification
* Interactive dashboard with spending insights
* Filter transactions by date, category, and type
* Export categorized data as CSV

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Scikit-learn
* Plotly
* Joblib

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt
   
2. Run the application:
   streamlit run app.py

3. Open **http://localhost:8501** in your browser.
